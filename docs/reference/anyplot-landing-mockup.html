§ 02

One import.

pip install anyplot

Same palette,
every library.

Every example in the catalogue uses the same Okabe-Ito palette. Switch libraries without losing your color grammar — a Gentoo penguin is always blue, whether you draw it in matplotlib or plotly.

Validated against deuteranopia, protanopia and tritanopia using the Machado et al. (2009) simulation model. 

# pick any library. the palette travels with you.
import anyplot as ap

data = ap.load("penguins")

# matplotlib
ap.mpl.scatter(data, x="bill", y="flipper",
               hue="species", palette="okabe-ito")

# plotly — same colors, interactive
ap.plotly.scatter(data, x="bill", y="flipper",
                  hue="species", palette="okabe-ito")

# list the palette explicitly
print(ap.palettes.okabe_ito)
# → ['#009E73', '#D55E00', '#0072B2', '#CC79A7',
#    '#E69F00', '#56B4E9', '#F0E442', adaptive]
